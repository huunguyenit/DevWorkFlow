# 09 — Generate ADR

---

**Prompt:**

Sinh ADR cho quyết định: **<mô tả>**

1. Đọc ADR có số lớn nhất hiện có trong repo để đặt số tiếp theo đúng thứ tự.
2. Dùng đúng cấu trúc: Context / Problem / Decision / Architecture (sơ đồ box-drawing) /
   Rationale / Consequences (Positive/Negative) / Alternatives Considered / Architectural Rules /
   Impact / Related Documents / Future Review.
3. Không mâu thuẫn 10 Core Principles trừ khi nêu rõ đây là thay đổi nền tảng (cảnh báo rõ).
4. Status ban đầu: `Proposed`, để tôi duyệt thành `Accepted`.
