# 10 — Generate Technical Specs

---

**Prompt:**

Đặc tả kỹ thuật cho: **<module/feature>**

Cấu trúc: Purpose, Scope/Non-goals, Vị trí trong kiến trúc, Data Model, Interface đề xuất,
Data Flow, Event, Extension Points, Performance targets, Test Plan, Rủi ro/Alternatives.
Ngắn gọn, dùng heading + bullet, không văn xuôi dài. Đọc `_SHARED/ARCHITECTURE_DOCTRINE.md`
mục 5 để xác định đúng module trước khi viết.
