# Product

Tên sản phẩm: FBO Studio

# Mục tiêu

Xây dựng một desktop application hỗ trợ lập trình viên triển khai và phát triển FBO ERP, giúp giảm thao tác lặp lại, tăng tốc phát triển và chuẩn hóa cấu hình.

# Đối tượng sử dụng

* Lập trình viên ERP
* Lập trình viên triển khai
* Technical Leader
* Hỗ trợ vận hành ERP

# Công nghệ dự kiến

* WPF
* MVVM
* Dependency Injection
* Dapper
* SQLite
* Serilog

# Tầm nhìn dài hạn

Phát triển thành IDE chuyên biệt cho FBO ERP, hỗ trợ:

* XML Designer
* SQL Studio
* Deployment Tool
* Workflow Designer
* Plugin System
* AI Assistant
* Multi ERP Version Support

# Nguyên tắc phát triển

1. Xây dựng từng module nhỏ nhưng sử dụng được ngay.
2. Mỗi phiên bản đều phải phục vụ công việc thực tế.
3. Không thiết kế quá mức (Over Engineering).
4. Dễ mở rộng và bảo trì trong ít nhất 10 năm.
5. Không hard-code metadata ERP.
6. Hỗ trợ nhiều phiên bản ERP thông qua cấu hình.

# Kiến trúc định hướng

* Clean Architecture (khi sản phẩm đủ lớn)
* MVVM
* Plugin Architecture
* Configuration Driven Development
