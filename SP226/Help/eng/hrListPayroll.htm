Payroll List

Salary Computing Period

Date from/ Date to

It is required to define the period of salary computing so that the system can gather all data being effective on this period to calculate salary, other income and PIT.

Regular Days

Regular number of working days in the salaried period. If this field is not defined, the system will get the "Working Days per Month" in optional parameter.

Salary Exchange Rate

- Salary exchange rate is used to exchange salary, award, penalty, allowance, tax in foreign currency into base currency (in case the employee is paid by foreign currency);
- The exchange rate used in the salaried period must be latest one before or on the last day of salaried period;
- Personal income tax is calculated in base currency and then exchanged to foreign currency.

Insurance Exchange Rate

- Insurance exchange rate is used to exchange insurance amount, union due into base currency (in case the employee is paid by foreign currency).
- The exchange rate used in the salaried period must be latest one before or on the last day of salaried period.

Minimum Wage Levels

Minimum wage is used to calculate actual salary (L00), base salary (B00) and other allowances (P20) of employees in case there is no region-based minimum wage.

Note: This common minimum wage will not used for employees paid by foreign currency, if there is no region-based minimum wage in foreign currency.

Region-based Minimum Wage Levels

Region-based minimum wage is used to calculate actual salary (L00), base salary (B00) and other allowances (P20) of employees.

Salary Category by Workday

Department

Extracted from Department List

Workday

Extracted from Workday List, only workday with "Salary Category" as "9 - Other"

Salary Category

Select the salary category to the selected workday (By time, product or piece rate)

For example:
Workday sign DH for training day with salary category 9 - Other;
Department BP1 calculated salary by time, the workday sign DH is assigned the salary category 1 – By Time;
Department BP2 calculated salary by product, the workday sign DH is assigned salary category 2 – Product;
Therefore, to employees having DH workday, ones in department BP1 will be calculated salary by time, ones in department BP2 will be calculated salary by product.

Allowance Type List

Used to classify allowances (in Allowance List)

Allowance List

Used to define information of each allowance such as the salary rate or allowance amount

Code

Code to identify allowance of employee

Short Name

Used to name the column title in allowance reports ("Employee Allowance Detailed Sheet")

Base Salary Rate (%)

Used for allowances calculated by base salary. Then, the allowance amount will be equal to this rate multiplied by base salary.

Amount

Used to input a fixed amount of allowance

Ordinal Number

Used to set order of allowance code viewed in report

Award, Penalty Reason List

Used to input "Awards, Penalties" information and view in "Employee Award, Penalty Detailed Sheet".

Reason

Code to identify award or penalty reason

Short Name

Used to name the column title in "Employee Award, Penalty Detailed Sheet"

Type

Used to determine this code is award or penalty

Income/Deduction List

Used to define the other income or deduction (select type 1 or 2) so that the system will add or deduct on salary

Award Type List

Used in case employees are rewarded by grade. The amount will be calculated by the coefficient multiplied by employee salary in period (excluding overtime pay).

Product List

Used to define product codes being used in salary calculated by product. The product codes will be used in Employee/Department Product Quantity Input.

Product Unit Cost

Used in case of calculating salary by product;
Product Salary = Product Unit Cost * Product Quantity

Coefficient by Department

Used to allocate department product salary to each employee
Product salary of each employee = Department Product Salary * Working hour coefficient of each employee / Total working hour coefficient of all employees in department.
In there: Working hour coefficient = number of working hours in working department x Workday Coefficient x Coefficient by Department.

Employee

Coefficient

Effective Date
